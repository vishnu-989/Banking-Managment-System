# Banking Management System

A comprehensive Bank Employee Management and Customer Banking web application built with React, Vite, and Tailwind CSS.

## Features

- **Customer Portal**: Account overview, fund transfers, bill payments, cheque services, FD/RD deposit opening, FX remittance, card management, loan applications, support tickets, tax certificates.
- **Employee Portal**: Customer onboarding, loan approvals, card issuance, customer support ticket resolution.
- **Admin Portal**: User & employee management, audit logging, fraud & risk center, product rates & system configuration.

## Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation & Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
